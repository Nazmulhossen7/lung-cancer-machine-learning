# Machine Learning Approaches for Early Detection of Lung Cancer: A Comparative Analysis

## Overview
This project investigates the effectiveness of various Machine Learning algorithms for the early detection of lung cancer. A comparative analysis was conducted using a publicly available lung cancer dataset to evaluate the predictive performance of different classification models.

## Objectives
- Compare the performance of multiple machine learning algorithms.
- Evaluate models using Accuracy, Precision, Recall, F1-Score, and ROC-AUC.
- Identify challenges such as overfitting and poor generalization.
- Provide recommendations for improving lung cancer prediction models.

## Dataset
Source: Kaggle Lung Cancer Dataset

Features:
- Age
- Gender
- Smoking
- Yellow Fingers
- Anxiety
- Peer Pressure
- Chronic Disease
- Fatigue
- Allergy
- Wheezing
- Alcohol Consuming
- Coughing
- Shortness of Breath
- Swallowing Difficulty
- Chest Pain

Target Variable:
- Lung Cancer (YES/NO)

## Machine Learning Models
